Install locally

  1. 1

    Build

    Compile the connectors once.

    ./scripts/cargo.ps1 build --release
  2. 2

    Register

    Point Claude Desktop at the binary with the snippet on each connector card.

  3. 3

    Restart

    Fully quit Claude Desktop (tray, then Exit) and reopen it. Ask Claude to use the new tools.
